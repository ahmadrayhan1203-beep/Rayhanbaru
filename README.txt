PKL Ahmad Rayhan - Multi-page static website (Version 2)

Pages included:
- index.html (Home)
- about.html (Tentang)
- activities.html (Kegiatan)
- gallery.html (Galeri + lightbox)
- contact.html (Kontak + mailto form)

Cara pakai:
1. Unzip dan upload isi folder ke root repository GitHub.
2. Aktifkan GitHub Pages di Settings -> Pages, pilih branch main dan folder root (/).
3. Akses di https://<username>.github.io/<repo>/

Catatan:
- Ganti nomor WhatsApp di contact.html dengan nomor asli jika ada.
- Ganti email di contact.html dengan email asli sebelum publikasi.
